# Fault Detection in Power Transmission Lines using Hybrid Deep Learning Models

## Overview
This project implements an end-to-end fault detection system for power transmission lines using multivariate time-series data.

## Project Structure
fault-detection-power-transmission/
├── pipeline/
├── models/
├── eda/
├── figures/
├── results/
├── requirements.txt
├── README.md
└── .gitignore

## Models Implemented
- BiGRU + Attention
- Transformer + MLP
- RNN + LightGBM
- LSTM Autoencoder + OC-SVM

## Evaluation
Models are evaluated using confusion matrices, ROC curves, and precision–recall curves.
